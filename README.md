# Random-Quote-Generator
Черноталов Михаил
GUI приложение для генерации случайных цитат с историей и фильтрацией
Можно использовать для вставки цитат в разного рода произведений, где они могут очень сильно понадобиться для более подробного описания той или иной ситуации, для описания персонажей или их реплики
